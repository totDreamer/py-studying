Функция choice() из модуля random возвращает случайный элемент из переданной ей последовательности, в роли которой может выступать строка, список, кортеж или любая другая коллекция.
Соответственно:
    импортируем функцию choice()из модуля random
    открываем заданный файл open('lines.txt', encoding='utf-8')
    применяем к открытому файлу метод file.readlines()и считываем в переменную список всех строк
    закрываем файл
    к переменной, в которой хранится весь список строк, применяем метод choice()и выводим эту случайную строку на печать при помощи print()
